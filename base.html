Base

Você pode usar o Waze CSS das duas seguintes formas:

Você pode chamar o Waze CSS completo.
Desta forma você ira utilizar todas as ferramentas oferecidas por ele, como Grids, Botões etc.

Você pode chamar os modulos Waze CSS separadamente.
Dentro da pasta de CSS existe uma pasta chamada modulos, onde estão separados os modulos do Waze CSS, utilizando dessa forma você pode chamar e usar somente a ferramenta que você precisa. Como por exemplo, se você chamar o modulo de grids seu projeto usará somente as grids e não os outros estilos.
