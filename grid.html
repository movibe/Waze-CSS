Grid

Introdução às Grids do Waze

Grids Waze são fáceis de trabalhar, e muito poderosas. Existem alguns conceitos simples para manter em mente:


Linha contra classes de Colunas
Grids Waze consistem em dois tipos de classes: Classe de linha ( row ) e classes de coluna ( col-* )


As larguras das colunas são frações
Colunas têm vários nomes de classe que representam suas larguras. Por exemplo, col-1-2 tem uma largura de 50%, enquanto col-1-5 teria uma largura de 20%.


Todos os elementos filho de uma linha deve ser coluna
Elementos filho contidos dentro de um elemento com um row deve ser uma coluna com um col-*.


Conteúdo vai dentro unidades de grade
Todo o conteúdo que é visível para as pessoas deve ser contido dentro de uma unidade row, isso garante que o conteúdo será processado corretamente.
